Keys:	Alt - Hide/Show
		Y - previous answer
		X - next answer
		Esc - close app

How to: Put Excel table with questions and answers in the same folder as svchost.exe  Excel table file must be called source  (source.xlsx)
		Excel table format: questions in first column and answers in second column, question and its answer(s) must be in the same row.
		Run svchost.exe and wait for window to show (program will be loading Excel data for a while).
		Select question to look for and copy it to clipboard (Ctrl + C), must be atleast 2 words.
		Then press Alt 2x to hide and show window again with new answers.
		Cycle through answers with Y and X keys.
